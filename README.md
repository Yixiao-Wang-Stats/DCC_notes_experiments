# DCC Notes Experiments

This repository is my personal learning and experiment-management workspace for the Duke CS computing cluster.

It contains:
- notes about login nodes, compute nodes, partitions, storage, and Slurm
- reusable experiment-management templates
- Slurm job templates
- environment notes
- troubleshooting logs
- records for small reproducible experiments

It does not contain:
- large datasets
- model checkpoints
- large experiment outputs
- caches
- passwords, tokens, or private credentials

Main learning goals:
1. Understand CPU/GPU nodes and partitions.
2. Learn a standard cluster workflow.
3. Deploy and manage TabPFN, GODST, and ZEUS experiments.
4. Monitor jobs, kernels, storage, and cleanup.
